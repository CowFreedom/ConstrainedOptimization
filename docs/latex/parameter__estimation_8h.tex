\doxysubsection{Detailed Description}
This file is used if the optimization functions are indirectly called. Indirectly means that the user only passes secondary information like path, options and shell commands instead of directly calling the optimization functions. The only work this file does is configuring the optimizers and reporting the status of the end result. Useful if the optimizers can be selected and configured based on user input.

Version number of the project. Higher numbers indicated more up-\/to-\/date versions. Contrary to the version counting procedure in e.\+g. Rust, the version number\textquotesingle{}s digits do not give an indication about the severity of the updates.
