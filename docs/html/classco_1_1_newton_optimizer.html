Detailed Description

template<class E>
class co::NewtonOptimizer< E >

This class represents an instance of the Newton Gauss algorithm. Instances of this class must be created if one wants to run the Newton Gauss algorithm. Instances of the Evaluator class from evaluator.h steer the computation behavior while instances of co::Options internally configure the class.
