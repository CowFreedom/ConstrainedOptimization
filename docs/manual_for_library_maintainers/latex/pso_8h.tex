\doxysubsection{Detailed Description}
The Particle Swarm Optimization (PSO) procedure minimizes functions. An explanation can be found in \char`\"{}\+Parameter Estimation in Ordinary Differential Equations Modeling via Particle Swarm Optimization\char`\"{} by Devin Akman et al. The algorithm scales well to multiple processing cores and is a useful precursor to find initial values for the Gauss Newton algorithm.
