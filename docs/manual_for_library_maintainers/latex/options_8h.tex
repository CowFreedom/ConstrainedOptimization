\doxysubsection{Detailed Description}
The only purpose of this file is to define settings and error codes that are used throughout the plugin. Optimizers are configured and report errors according to the enums and classes defined in this header. Noteworthy are the co\+::\+Config\+Output and co\+::\+Config\+Evaluation enums, as they are problem dependent and might significantly alter the behavior of the optimizers.
